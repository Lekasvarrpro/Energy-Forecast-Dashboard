# Energy Forecast Dashboard

A responsive and interactive web dashboard that provides daily, monthly, and yearly energy forecasts using real CSV data. It visualizes energy consumption trends, highlights peak usage, and offers appliance-wise breakdowns, helping users make data-driven decisions for efficient energy management. Fully deployed on **Vercel**.

---

## 1. Overview

The **Energy Forecast Dashboard** is a web-based application built to analyze and visualize energy consumption data. Users can explore forecasts across different time frames (day, month, year) through intuitive visualizations, including calendar heatmaps and detailed appliance-wise breakdowns. The dashboard leverages real CSV-based datasets for accurate and dynamic energy insights.

---

## 2. Tools & Technologies

- **Frontend Framework:** React + Vite  
- **Styling:** Tailwind CSS  
- **Data Processing:** PapaParse for CSV parsing  
- **Visualization:** Custom calendar heatmaps and tables  
- **Deployment:** Vercel  
- **Icons & UI Enhancements:** React Icons, Framer Motion  
- **Version Control:** Git + GitHub  

---

## 3. How It Works

1. **Data Loading:**  
   The dashboard reads energy data from a CSV file located in the `public` folder.

2. **Dynamic Views:**  
   - **Day View:** Shows detailed energy usage per appliance for the selected day.  
   - **Month View:** Displays a calendar heatmap showing daily energy consumption trends.  
   - **Year View:** Summarizes energy usage per month with color-coded blocks and forecasts.

3. **Interactivity:**  
   Users select the date or year, choose the forecast type (day/month/year), and click **Generate** to display the results. Smooth animations and color gradients enhance visualization clarity.

4. **Energy Insights:**  
   High usage is highlighted in red, moderate usage in yellow/orange, and low usage in green. Future months are marked with ‚ÄúFuture Update‚Äù for planning.
